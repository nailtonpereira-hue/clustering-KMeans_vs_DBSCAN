# K-Means vs DBSCAN

**Universidade Federal da Paraíba - UFPB**  
**Curso de Licenciatura em Ciência da Computação**  
**Disciplina de Inteligência Artificial – 2026-1**

---
Para o projeto final da **Unidade III**, planejamos fazer uma análise comparativa de dois algoritmos de agrupamento, **K-Means** e **DBSCAN**, para diferenciar características de digitação em um dataset de **Keyboard Analytics**.  O objetivo deste trabalho é comparar a capacidade de cada **algoritmo** em agrupar corretamente padrões de digitação pertencentes a diferentes indivíduos no mesmo **cluster**, investigando a viabilidade do uso de técnicas de agrupamento para a diferenciação de usuários com base em características de **Keyboard Analytics**. Para isso, serão utilizados dados disponíveis em uma base de dados **open access** e, se necessário, novos dados de digitação serão coletados.
